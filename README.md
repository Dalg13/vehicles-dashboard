# Vehicles Dashboard Project

Este proyecto es una aplicación web interactiva construida con **Streamlit**...",

El dataset utilizado proviene del archivo "vehicles_us.csv" y contiene datos de distintos vehículos usados listados para su venta.

El objetivo de este dashboard es para hacer un análisis exploratorio del listado de vehículos.

Para ejecutar la app localmente se tiene que usar el siguiente código:
  python3 -m venv venv
  source venv/bin/activate      # macOS / Linux
  venv\Scripts\activate         # Windows
  pip install -r requirements.txt
  streamlit run app.py


URL del despliegue es el siguiente: https://vehicles-dashboard-q19f.onrender.com/
